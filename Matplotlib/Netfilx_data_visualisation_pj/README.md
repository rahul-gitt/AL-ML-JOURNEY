# 🎬 Netflix Data Analysis & Visualization

## 📌 Project Overview

This project explores the Netflix Movies & TV Shows dataset using Python, Pandas, and Matplotlib.

The goal is to perform Exploratory Data Analysis (EDA) and create meaningful visualizations to understand trends in Netflix content.

---

## 📂 Dataset

- Netflix Movies and TV Shows Dataset
- Total Records: 8,807
- Features: 12

---

## 🛠 Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook

---

## 📊 Analysis Performed

✔ Data Cleaning

✔ Missing Value Analysis

✔ Movies vs TV Shows Comparison

✔ Content Released by Year

✔ Top 10 Countries with Most Netflix Content

✔ Ratings Analysis

✔ Genre Analysis

✔ Duration Distribution

---

## 📈 Visualizations

- Bar Chart
- Horizontal Bar Chart
- Line Chart
- Pie Chart
- Histogram

---

## 🔍 Key Insights

- The United States has the highest amount of Netflix content.
- India is the second-largest content producer.
- Netflix content increased rapidly after 2015.
- Movies significantly outnumber TV Shows.
